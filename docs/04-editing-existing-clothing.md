# Editing Existing Clothing

Editing existing clothing is the safest and fastest way to start.

## Workflow
1. Export `.ydd` and `.ytd` from OpenIV
2. Import using Sollumz
3. Edit mesh in Edit Mode only
4. Adjust UVs if needed
5. Re-export

## Common Mistakes
- Applying transforms
- Breaking vertex groups
- Changing bone names
